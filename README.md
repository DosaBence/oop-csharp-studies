# oop-csharp-studies
A collection of C# console applications focused on core Object-Oriented Programming (OOP) principles and file input/output operations.


Polymorphism & OOP Architecture: Implementation of dynamic polymorphism using virtual methods, overriding, and object-oriented class structures (Console_classes_polimorfism.cs).
File Input Operations: Automated reading and parsing of external data streams from text files (Console_file_read.cs).
Data Parsing & Field Structuring: Extracting and managing data structures containing 4 distinct fields from text files (Console_file_4fields.cs).
Advanced Data Processing: Complex file handling and structured field management (Console_file_4fields_complex.cs).


Language: C#
Framework: .NET Console Application

Concepts: Object-Oriented Programming (OOP), Polymorphism, Encapsulation, File I/O (StreamReader/StreamWriter), String Parsing.

Clone the repository or download the source files.
Open the desired .cs file in an IDE (e.g., Visual Studio or Visual Studio Code).
Ensure target text files for reading/writing operations are placed in the correct build directory.
Compile and run the application.
